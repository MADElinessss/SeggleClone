# Seggle_api
This project is to imitate [Seggle backend](github.com/seggle/BE).

## Installation
Djangogram과 유사합니다.

### Environment
* Python 3.10 or later
* PyCharm (Optional)

### Required pip packages
* Django (>= 4.0)
* djangorestframework
* djangorestframework-simplejwt
* mysqlclient
* Pillow
